# StayHealthy - Medical Appointment Booking (Frontend)

StayHealthy is a frontend web application for searching doctors by specialty, booking and cancelling appointments, and submitting reviews for completed consultations.

## Features
- Responsive navigation bar (Appointments, Reviews, Sign Up, Login)
- Sign Up and Login forms with basic validation messages
- Doctor search by specialty
- Doctor cards with details (name, specialty, experience, rating) and booking action
- Appointment booking and cancellation flow
- Reviews section with rating + feedback form (one review per consultation)

## Tech Stack
- React
- TypeScript
- CSS (component-based styles)
- Local Storage (for persisting simple session/appointment/review data)
